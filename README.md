#MyIRC - Epitech Project (PSU Network)

Create an IRC Server and Client
The project have to be compliant with the RFC 1490

## Compile
```bash
make
```

## Usage
```bash
# For the server
./server (port) # The server Listening port
# For teh client
./client
```

## Unit Tests
```bash
make unit_tests
```

## Documentation
```bash
make doxygen
```
